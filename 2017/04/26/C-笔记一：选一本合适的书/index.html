前言

学一门编程语言，就从最难的开始，学会了意味着收获最大的成就感，也意味着困难重重，选中了C++, 是因为我本身是ios程序猿，OC 继承自C++, 一脉相承，我学起来应该没有传说中的那么难。

学一门语言，首先要选择一本书，一本适合自身情况的书，每个人的情况不一样，对每个人来说，适合自身的书自然也不一样，对别人好的书，不一定适合自己，所以不要去百度，google 最好的C++教材。

我选择的是《C++ from scratch》作者：Jesse Liberty 中文译本《C++ 编程实例详解》 在1999年出版过一个版本，所以现在中文版的书很难买到了，并且译文有很多处错误，不过英文版的在亚马逊和eBay上有售，而且英文版随书所带的光盘中代码也有很多错误，不过书确实不失为一本好书，后面会附上修正过的源码，由于书从美国邮寄，所以有些贵。

这本书通过完善一个名字为《Decryptix》的游戏，来讲解程序流，类，指针，宏，链表，重载，多态，STL模版，异常处理。没有过多讲解语法，不着中某一技术难点，所以这本书不适合一门编程语言都没掌握的新手。

修正过的decrytix游戏代码地址：https://github.com/toct/Decryptix.git
