最近求职，发现很多公司要求提供自己的技术博客，可能是有博客的人都是大牛，所以为了步入大牛行列，面子工程是要做一做的。据说长期写博客 记录工作，学习，能够让你的文笔更好，学的更快，记得更牢，真的这么神奇么？我决定先坚持一年，亲身体验一下。

对于如我一样没有安全感的人，自己的东西都要带在身上，看的见摸的着，心理才踏实，hexo+github这种形式写博客是一个不错的选择，hexo一款基于Node.js静态博客，在本地生成静态网页，在github上创建一个repository来托管这些网页，下面上干货：

Quick Start

配置环境

使用工具 MACBook(版本：10.12.4) 
Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境，Node.js 的包管理器 是npm

在安装hexo期间会用到很多以npm开头的命令，去Node网站下载最新版本，一路傻瓜式安装。

安装Hexo

这是一个快捷，简单，又非常强大的博客框架。
创建一个文件夹,如hexo,用户存放hexo的配置文件，终端cd到hexo目录下面

应该可以看到页面上有一篇hellow world 的文章，说明 你hexo安装成功了

执行命令：npm install hexo-deployer-git –save
然后上传到github上：hexo deploy
常用命令：hexo n “我的博客” == hexo new “我的博客” #新建文章
hexo p == hexo publish
hexo g == hexo generate#生成
hexo s == hexo server #启动服务预览
hexo d == hexo deploy#部署

更换hexo主题

网上有各种样式的已经写好hexo主题，我选的是Kiddochan 大家也可以选择去hexo选择中意的主题

cd到hexo目录下执行拷贝命令：git clone https://github.com/hsihohuang/kiddochan.git themes/Kiddochan
也可以直接去github上下载其它的主题拷贝到hexo／themes 下

然后vim _config.yml 修改主题名字为你选择的主题名字：theme: Kiddochan
然后保存退出

部署：npm install hexo-deployer-git –save
然后发布：hexo clean && hexo g && hexo d
